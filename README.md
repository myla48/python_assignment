## 👤 Author

## 📌 Description
This repository contains solutions for the Python Foundations daily assignment.

The project includes three tasks from different levels:
- Level 1: Reverse a string
- Level 2: Count vowels in a string
- Level 3: CLI-based calculator

These programs demonstrate basic Python concepts such as:
- Functions
- Loops
- Conditionals
- User input handling

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run each file using:

```bash
python level1_reverse.py
python level2_vowels.py
python level3_calculator.py
